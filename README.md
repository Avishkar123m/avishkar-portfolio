# AVISHKAR SANJEEWAN MANDHARE — Portfolio

This folder is a standalone, static website generated from your resume. It is
completely independent from the portfolio generator app — there is no build
step and no framework required.

## View it locally

Just open `index.html` in your browser.

## Deploy it

- **Vercel / Netlify:** drag-and-drop this folder onto vercel.com/new or
  app.netlify.com/drop, or push it to a GitHub repo and import it — both
  auto-detect it as a static site, no configuration needed.
- **GitHub Pages:** push this folder to a repo and enable Pages for it in
  Settings → Pages. Since this is plain HTML/CSS/JS, it works with no changes.

## Structure

```
index.html      Your portfolio page
css/styles.css  All styling
js/script.js    Small vanilla-JS helpers (theme toggle, back-to-top, mobile nav)
Avishkar_Mandhare_Resume_260712_022225 (2).docx   Your original uploaded resume (downloaded by the "Download Resume" button)
```
